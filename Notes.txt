# Formatting Markdown Language

# =====================================================================================================================

# Here is a list of some common formatting options available in Markdown language:
#
# Bold: To create bold text, you can use double asterisks (**) or double underscores (__) before and after the text. For
# example: bold text or bold text
#
# Italic: To create italicized text, you can use single asterisks (*) or underscores (_) before and after the text. For
# example: italicized text or italicized text
#
# Strikethrough: To create strikethrough text, you can use double tildes (~~) before and after the text. For example:
# strikethrough text
#
# Code: To create inline code blocks, you can use single backticks () before and after the text. For example: code`
#
# Code blocks: To create code blocks, you can use triple backticks (```) before and after the text.
#
# Links: To create a link, you can use square brackets ([]) for the link text and parentheses (()) for the link URL.
# For example: Google
#
# Headings: To create headings, you can use pound signs (#) before the heading text. The number of pound signs indicates
# the heading level, with one pound sign representing a top-level heading and six pound signs representing a
# sixth-level heading.
#
# Lists: To create a bulleted list, you can use asterisks (*) or hyphens (-) before each list item. To create a
# numbered list, you can use numbers followed by a period (.) before each list item.
#
# Horizontal lines: To create a horizontal line, you can use three or more asterisks (***), hyphens (---), or
# underscores (___) on their own line.
#
# Blockquotes: To create a blockquote, you can use the greater than sign (>) before the text.
